HTML is the standard language used for displaying documents in a web browser. It's elements are the building blocks of most web pages on the internet and is usually used alongside technologies such as Cascading Style Sheets (CSS) and JavaScript

It's easy to learn HTML, there is an abundance of free resources online to get started. Three of my personal favourites can be found below
